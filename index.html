Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • An element includes opening and closing tags as well as the content between them.
  • Elements include headers, paragraphs, unordered & ordered lists, list items, images, line breaks, and anchors.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which the rules (declarations) will apply.
  • Declarations contain the CSS property that will be applied and the value for that property (div {property: value})
  • An external CSS file must be linked to the HTML file using a "link" element.
  • A class attribute allows us to share a CSS rule to any element using a class selector such as ".card"

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls any updated code on main branch
  • git add -A: adds all changes
  • git commit: submits commit on current branch
