HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • An element includes opening and closing tags as well as the content between them.
  • Elements include headers, paragraphs, unordered & ordered lists, list items, images, line breaks, and anchors.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which the rules (declarations) will apply.
  • Declarations contain the CSS property that will be applied and the value for that property (div {property: value})
  • An external CSS file must be linked to the HTML file using a "link" element.
  • A class attribute allows us to share a CSS rule to any element using a class selector such as ".card"

Git

  • "Git Flow" refers to the steps needed to commit, pull, and merge your working branch into the main branch and use git version Control.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all changes; this stages any changes you've made to the code
  • git commit -m "descriptive comment about changes made"
  • git pull origin main: pulls any updated code from the main branch
  • git push origin [branch name]
  • The rest of the Git Flow happens on github.com
  • git checkout main: navigates to the main branch in the terminal
  • git pull origin main

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • An array is a group of related data that act as a variable.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional statements, [for] loops, and functions are ways to change control flow.
  • "Commenting out" makes a section of code inactive without deleting it.
  • [for] loops contain three statements: 1-indicate the starting point for the loop, 2-name a condition, 3-allow re-iterations.
  • A function is a set of instructions that tell a computer how to perform a certain task. This allows us to define and reuse blocks of code.
  • To use a function you must 1-define the function, 2-call the function.
  • A function declaration uses the [function] keyword and the name of the function: ex. function makeTea().
  • To define a function you must have a function declaration followed by a {code block}
  • To call a function enter the function name followed by ().
